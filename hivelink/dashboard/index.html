Setup guide
Install HiveLink on every machine you want in the cluster — same steps on all platforms
Installation
1
Install Python 3.11+
Required on every node. Python 3.12 recommended.
python3 --version # must be ≥ 3.11
2
Install HiveLink
Same command on Windows, Linux, and macOS.
pip install hivelink # NVIDIA GPU support pip install "hivelink[nvidia]" # AMD GPU support pip install "hivelink[amd]"
3
Install llama.cpp
Downloads the prebuilt llama-server binary for your platform.
hivelink install-llama
4
Start on every machine
Run this on each device — they discover each other automatically within a few seconds. No config needed.
hivelink start
5
Download a model and run it
Models from Ollama, LM Studio, or Hugging Face all work — they're all GGUF format under the hood.
# Download any GGUF model from Hugging Face huggingface-cli download bartowski/Llama-3.1-8B-Instruct-GGUF \ --include "Llama-3.1-8B-Instruct-Q4_K_M.gguf" # Or use a model Ollama already downloaded # Ollama stores GGUF files at: ~/.ollama/models/blobs/ # Start llama-server with your model llama-server --model path/to/model.gguf --port 8080
