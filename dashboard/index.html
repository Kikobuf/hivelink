Setup
Install HiveLink on every machine you want in the cluster
Installation
1
Install Python 3.11+
Required on every node.
python3 --version # must be ≥ 3.11
2
Install HiveLink
Same command on all platforms.
pip install hivelink # NVIDIA GPU: pip install "hivelink[nvidia]" # AMD GPU: pip install "hivelink[amd]"
3
Start on every machine
Nodes discover each other automatically. No config needed.
hivelink start
4
Start Ollama or llama-server
HiveLink routes chat requests to your local inference engine.
# Ollama (easiest) ollama run llama3.2 # llama-server direct llama-server --model path/to/model.gguf --port 8080 # Set backend port (Ollama=11434, llama-server=8080) set LLAMA_CPP_PORT=11434 # Windows export LLAMA_CPP_PORT=11434 # Mac/Linux
