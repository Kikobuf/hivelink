Example — Python
Setup
Install HiveLink on every machine in the cluster
Installation
1
Install Python 3.11+
Required on every node.
python3 --version # must be ≥ 3.11
2
Install HiveLink
Same command on all platforms.
pip install hivelink # NVIDIA GPU: pip install "hivelink[nvidia]" # AMD GPU: pip install "hivelink[amd]"
3
Start on every machine
Nodes discover each other automatically.
hivelink start
4
Start Ollama or llama-server
HiveLink auto-detects whichever is running.
ollama run llama3.2 # Ollama (port 11434) llama-server --model model.gguf # llama-server (port 8080)

Downloads via Ollama on this node. Appears in the cluster once pulled.
