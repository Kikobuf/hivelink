Setup guide
Install HiveLink on every machine you want in the cluster
Installation
1
Install Python 3.11+
Required on every node.
python3 --version # must be ≥ 3.11
2
Install HiveLink
Same command on all platforms.
pip install hivelink # NVIDIA GPU pip install "hivelink[nvidia]" # AMD GPU pip install "hivelink[amd]"
3
Start on every machine
Nodes discover each other automatically via UDP broadcast. No config needed.
hivelink start
4
Start Ollama or llama-server with a model
HiveLink routes chat requests to your local inference engine.
# Option A — Ollama (easiest) ollama run llama3.2 # Option B — llama-server directly llama-server --model path/to/model.gguf --port 8080 # Tell HiveLink which port your inference engine is on # Ollama: port 11434 | llama-server default: port 8080 set LLAMA_CPP_PORT=11434 # Windows export LLAMA_CPP_PORT=11434 # Mac/Linux
